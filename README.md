# Analisis-Datos-Transporte-UM
Este repositorio analiza el transporte de última milla usando una arquitectura de medallón (Bronze, Silver, Gold). Incluye limpieza y modelamiento de datos, desarrollo de dashboard en Power BI con KPIs clave y generación de conclusiones para optimizar la operación logística.
